SnapshotJob Behavior by Source and Tier

  Tier 1: Staleness Check

  ┌─────────────────────┬──────────────────────────────┬────────────────────────────────────────────────┬───────────────────────────┐
  │       Source        │       Staleness check?       │                  How checked                   │         If stale          │
  ├─────────────────────┼──────────────────────────────┼────────────────────────────────────────────────┼───────────────────────────┤
  │ Poll                │ Yes                          │ newest.Age > IntervalSeconds × GraceMultiplier │ Skip to tier 4 (commands) │
  ├─────────────────────┼──────────────────────────────┼────────────────────────────────────────────────┼───────────────────────────┤
  │ Synthetic           │ Yes (if IntervalSeconds > 0) │ Same formula                                   │ Skip to tier 4 (commands) │
  ├─────────────────────┼──────────────────────────────┼────────────────────────────────────────────────┼───────────────────────────┤
  │ Trap                │ Excluded                     │ —                                              │ —                         │
  ├─────────────────────┼──────────────────────────────┼────────────────────────────────────────────────┼───────────────────────────┤
  │ Command             │ Excluded                     │ —                                              │ —                         │
  ├─────────────────────┼──────────────────────────────┼────────────────────────────────────────────────┼───────────────────────────┤
  │ IntervalSeconds=0   │ Excluded                     │ —                                              │ —                         │
  ├─────────────────────┼──────────────────────────────┼────────────────────────────────────────────────┼───────────────────────────┤
  │ No data (null slot) │ Skipped                      │ —                                              │ Not considered stale      │
  └─────────────────────┴──────────────────────────────┴────────────────────────────────────────────────┴───────────────────────────┘

  If ANY non-excluded holder is stale → entire tenant skips to tier 4.
  If NONE stale → proceed to tier 2.

  Tier 2: Resolved Gate

  ┌─────────────────────┬────────────────────┬──────────────────────────────┬──────────────────────────────┬─────────────────────┐
  │       Source        │  Samples checked   │        Violated means        │   If ALL resolved violated   │ If ANY not violated │
  ├─────────────────────┼────────────────────┼──────────────────────────────┼──────────────────────────────┼─────────────────────┤
  │ Poll                │ All series samples │ Every sample fails threshold │ Return Violated, no commands │ Continue to tier 3  │
  ├─────────────────────┼────────────────────┼──────────────────────────────┼──────────────────────────────┼─────────────────────┤
  │ Synthetic           │ All series samples │ Every sample fails threshold │ Return Violated, no commands │ Continue to tier 3  │
  ├─────────────────────┼────────────────────┼──────────────────────────────┼──────────────────────────────┼─────────────────────┤
  │ Trap                │ Newest only        │ Newest fails threshold       │ Return Violated, no commands │ Continue to tier 3  │
  ├─────────────────────┼────────────────────┼──────────────────────────────┼──────────────────────────────┼─────────────────────┤
  │ Command             │ Newest only        │ Newest fails threshold       │ Return Violated, no commands │ Continue to tier 3  │
  ├─────────────────────┼────────────────────┼──────────────────────────────┼──────────────────────────────┼─────────────────────┤
  │ Empty series        │ Skipped            │ Does not participate         │ —                            │ —                   │
  ├─────────────────────┼────────────────────┼──────────────────────────────┼──────────────────────────────┼─────────────────────┤
  │ No Resolved holders │ Vacuous true       │ —                            │ Return Violated              │ —                   │
  └─────────────────────┴────────────────────┴──────────────────────────────┴──────────────────────────────┴─────────────────────┘

  Tier 3: Evaluate Gate

  ┌───────────────────────────────┬────────────────────┬──────────────────────────────┬──────────────────────────┬───────────────────────────────────────┐
  │            Source             │  Samples checked   │        Violated means        │ If ALL evaluate violated │          If ANY not violated          │
  ├───────────────────────────────┼────────────────────┼──────────────────────────────┼──────────────────────────┼───────────────────────────────────────┤
  │ Poll                          │ All series samples │ Every sample fails threshold │ Proceed to tier 4        │ Return Healthy, no commands           │
  ├───────────────────────────────┼────────────────────┼──────────────────────────────┼──────────────────────────┼───────────────────────────────────────┤
  │ Synthetic                     │ All series samples │ Every sample fails threshold │ Proceed to tier 4        │ Return Healthy, no commands           │
  ├───────────────────────────────┼────────────────────┼──────────────────────────────┼──────────────────────────┼───────────────────────────────────────┤
  │ Trap                          │ Newest only        │ Newest fails threshold       │ Proceed to tier 4        │ Return Healthy, no commands           │
  ├───────────────────────────────┼────────────────────┼──────────────────────────────┼──────────────────────────┼───────────────────────────────────────┤
  │ Command                       │ Newest only        │ Newest fails threshold       │ Proceed to tier 4        │ Return Healthy, no commands           │
  ├───────────────────────────────┼────────────────────┼──────────────────────────────┼──────────────────────────┼───────────────────────────────────────┤
  │ Empty series                  │ Skipped            │ Does not participate         │ —                        │ —                                     │
  ├───────────────────────────────┼────────────────────┼──────────────────────────────┼──────────────────────────┼───────────────────────────────────────┤
  │ No Evaluate holders with data │ Vacuous false      │ —                            │ —                        │ Return Healthy (no data = no command) │
  └───────────────────────────────┴────────────────────┴──────────────────────────────┴──────────────────────────┴───────────────────────────────────────┘

  Tier 4: Command Dispatch

  ┌─────────────────────────────────────┬──────────────────────────────────────────────────┬───────────┐
  │              Condition              │                     Behavior                     │  Result   │
  ├─────────────────────────────────────┼──────────────────────────────────────────────────┼───────────┤
  │ Command not suppressed              │ Enqueue to channel                               │ Commanded │
  ├─────────────────────────────────────┼──────────────────────────────────────────────────┼───────────┤
  │ Command suppressed                  │ Increment snmp_command_suppressed_total          │ Skip      │
  ├─────────────────────────────────────┼──────────────────────────────────────────────────┼───────────┤
  │ Channel full                        │ Increment snmp_command_failed_total, log warning │ Skip      │
  ├─────────────────────────────────────┼──────────────────────────────────────────────────┼───────────┤
  │ All commands enqueued (count > 0)   │ —                                                │ Commanded │
  ├─────────────────────────────────────┼──────────────────────────────────────────────────┼───────────┤
  │ Zero enqueued (all suppressed/full) │ —                                                │ Violated  │
  └─────────────────────────────────────┴──────────────────────────────────────────────────┴───────────┘

  Advance Gate (priority group scope)

  ┌───────────┬───────────────────────────────────┐
  │  Result   │ Advances to next priority group?  │
  ├───────────┼───────────────────────────────────┤
  │ Healthy   │ Yes                               │
  ├───────────┼───────────────────────────────────┤
  │ Violated  │ Yes                               │
  ├───────────┼───────────────────────────────────┤
  │ Commanded │ No — blocks lower-priority groups │
  └───────────┴───────────────────────────────────┘

  ┌────────────────────────────────────────────────┬────────────────┬──────────────────┬──────────────────┬─────────────────────┬───────────┐
