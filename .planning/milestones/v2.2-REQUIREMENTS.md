## v2.2 Requirements

Requirements for Progressive E2E Snapshot Suite milestone. Each maps to roadmap phases.

### Stage 1: Single Tenant Evaluation States

- [x] **PSS-01**: Not Ready state detected before grace window ends (1 tenant, group 1)
- [x] **PSS-02**: Stale poll data triggers tier=1 then tier=4 command dispatch
- [ ] **PSS-03**: Stale synthetic source triggers tier=1 then tier=4 command dispatch — DEFERRED (requires synthetic-sourced OID in tenant fixture)
- [ ] **PSS-04**: Trap-sourced holder unaffected by stale poll data (immunity) — DEFERRED (requires trap-sourced OID in tenant fixture)
- [ ] **PSS-05**: Command-sourced holder unaffected by stale poll data (immunity) — DEFERRED (requires command-sourced OID in tenant fixture)
- [x] **PSS-06**: All resolved violated produces tier=2 Resolved, zero commands
- [x] **PSS-07**: Partial resolved violated continues to tier=3 (not tier=2)
- [x] **PSS-08**: All evaluate violated produces tier=4 Unresolved, commands dispatched
- [x] **PSS-09**: All metrics in-range produces tier=3 Healthy, no action
- [x] **PSS-10**: Unresolved with suppression — tier=4 but commands suppressed within window

### Stage 2: Two Tenant Independence

- [x] **PSS-11**: T1=Healthy + T2=Unresolved — independent per-tenant results, no interference
- [x] **PSS-12**: T1=Resolved + T2=Healthy — both evaluated with correct per-tenant tier logs
- [x] **PSS-13**: Both tenants Unresolved — both dispatch commands independently

### Stage 3: Advance Gate Logic

- [x] **PSS-14**: All G1 Resolved — gate passes, G2 evaluated
- [x] **PSS-15**: All G1 Healthy — gate passes, G2 evaluated
- [x] **PSS-16**: G1 mixed Resolved+Healthy — gate passes, G2 evaluated
- [x] **PSS-17**: All G1 Unresolved — gate blocks, G2 not evaluated
- [x] **PSS-18**: G1 mixed Resolved+Unresolved — gate blocks, G2 not evaluated
- [x] **PSS-19**: G1 mixed Healthy+Unresolved — gate blocks, G2 not evaluated
- [x] **PSS-20**: All G1 Not Ready — gate blocks, G2 not evaluated

### Infrastructure

- [x] **PSS-INF-01**: Stage gating — Stage 2 runs only if Stage 1 passes, Stage 3 only if Stage 2 passes
- [x] **PSS-INF-02**: Three tenant fixtures (1-tenant, 2-tenant, 4-tenant) with clean configmap restore
- [x] **PSS-INF-03**: Reuses existing OIDs, simulator HTTP endpoints, sim.sh helpers
