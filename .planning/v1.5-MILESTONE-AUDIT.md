## E2E Flow

**ConfigMap → Watcher → Registry → Pipeline → Counter**

1. Operator applies simetra-tenantvector ConfigMap
2. TenantVectorWatcherService detects via K8s API watch
3. TenantVectorOptionsValidator checks config against OID map
4. TenantVectorRegistry.Reload() atomically rebuilds routing index (volatile swap)
5. TenantVectorFanOutBehavior routes matching SNMP samples to tenant slots
6. snmp_tenantvector_routed_total counter increments (Prometheus-observable)

**Status: Complete — no breaks in chain**
