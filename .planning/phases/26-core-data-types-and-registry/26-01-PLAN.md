    Create five files in `src/SnmpCollector/Pipeline/`, all in the `SnmpCollector.Pipeline` namespace.

    **MetricSlot.cs:**
    - `public sealed record MetricSlot(double Value, string? StringValue, DateTimeOffset UpdatedAt);`
    - MUST be `record` (class, NOT `record struct`) — reference type for atomic pointer swap.
    - XML doc: "Immutable value cell for a single metric observation. Swapped atomically via Volatile.Write in MetricSlotHolder."

    **MetricSlotHolder.cs:**
    - `public sealed class MetricSlotHolder` with:
      - `private volatile MetricSlot? _slot;` — starts null (no value yet)
      - Readonly properties: `string Ip`, `int Port`, `string MetricName`, `int IntervalSeconds`
      - Constructor: `(string ip, int port, string metricName, int intervalSeconds)`
      - `public void WriteValue(double value, string? stringValue)` — creates new MetricSlot with `DateTimeOffset.UtcNow` and calls `Volatile.Write(ref _slot, newSlot)`
      - `public MetricSlot? ReadSlot()` — returns `Volatile.Read(ref _slot)`
    - Declare field as `volatile` AND use Volatile.Read/Write in methods (belt and suspenders per RESEARCH).

    **RoutingKey.cs:**
    - `public readonly record struct RoutingKey(string Ip, int Port, string MetricName) : IEquatable<RoutingKey>;`
    - `public sealed class RoutingKeyComparer : IEqualityComparer<RoutingKey>` with:
      - `public static readonly RoutingKeyComparer Instance = new();`
      - `Equals`: Port == Port AND string.Equals(Ip, Ip, OrdinalIgnoreCase) AND string.Equals(MetricName, MetricName, OrdinalIgnoreCase)
      - `GetHashCode`: `HashCode.Combine(OrdinalIgnoreCase.GetHashCode(Ip), Port, OrdinalIgnoreCase.GetHashCode(MetricName))`
    - Do NOT override the struct's default Equals/GetHashCode — the comparer is passed to FrozenDictionary explicitly.

    **Tenant.cs:**
    - `public sealed class Tenant` with readonly properties:
      - `string Id { get; }`
      - `int Priority { get; }`
      - `IReadOnlyList<MetricSlotHolder> Holders { get; }`
    - Constructor: `(string id, int priority, IReadOnlyList<MetricSlotHolder> holders)`

    **PriorityGroup.cs:**
    - `public record PriorityGroup(int Priority, IReadOnlyList<Tenant> Tenants);`
    - Simple named record, not sealed (records cannot be sealed in C#).
