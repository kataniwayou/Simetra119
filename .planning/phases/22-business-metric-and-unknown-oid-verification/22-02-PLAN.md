```
# Scenario 15: Unmapped OIDs classified as metric_name="Unknown"
# Adds unmapped OIDs to E2E-SIM poll config, waits for them to appear
# as Unknown in Prometheus, then restores original ConfigMap.
```

Steps:
1. Call `snapshot_configmaps` to backup current state
2. Apply `$SCRIPT_DIR/fixtures/e2e-sim-unmapped-configmap.yaml` via `kubectl apply -f ... -n simetra`
3. Wait for data to appear: use `poll_until_exists` with metric query or manual polling loop with 45s timeout, 3s interval. Poll for `snmp_gauge{device_name="E2E-SIM",metric_name="Unknown"}` to have results > 0.

     ```bash

         sleep 3
     done
     ```
