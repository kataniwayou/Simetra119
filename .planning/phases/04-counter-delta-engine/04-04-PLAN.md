  <behavior>
    The CounterDeltaEngine computes correct deltas for all counter scenarios:

    Test cases (input -> expected output):

    1. First poll: RecordDelta(oid, agent, "poll", "metric", Counter32, 1000, uptime=50000)
       -> returns false, CounterRecords empty (baseline stored, nothing emitted)

    2. Normal increment: baseline=1000, then RecordDelta(..., Counter32, 1500, uptime=60000)
       -> returns true, CounterRecords[0].Delta == 500.0

    3. Counter32 wrap: baseline=4_294_967_200, then RecordDelta(..., Counter32, 100, uptime=60000)
       -> returns true, delta == (2^32 - 4294967200) + 100 == 196.0

    4. Reboot detected (sysUpTime decreased): baseline with uptime=90000, then RecordDelta(..., Counter32, 300, uptime=1000)
       -> returns true, delta == 300.0 (current value used as delta)

    5. Counter64 decrease (always reboot): baseline=5000, then RecordDelta(..., Counter64, 200, uptime=60000)
       -> returns true, delta == 200.0

    6. Two agents same OID: agent-a baseline=1000, agent-b baseline=5000, agent-a second=1100, agent-b second=5050
       -> CounterRecords[0].Delta == 100.0 (agent-a), CounterRecords[1].Delta == 50.0 (agent-b)

    7. sysUpTime null with current < previous (conservative reboot): baseline=5000 with uptime=null, then current=300 with uptime=null
       -> delta == 300.0 (reboot assumed)

    8. Counter32 wrap at exact boundary: baseline = 4_294_967_295 (2^32-1), current = 0
       -> delta == (2^32 - 4294967295) + 0 == 1.0

    9. Same value (zero delta): baseline=1000, then current=1000
       -> delta == 0.0 (valid, emitted)

    10. RecordDelta label pass-through: verify metricName, oid, agent, source in CounterRecords tuple
  </behavior>
  <implementation>
    Tests instantiate CounterDeltaEngine directly with TestSnmpMetricFactory and NullLogger.
    No DI container, no MediatR, no SNMP network I/O.

    Test class structure:
    - Private fields: TestSnmpMetricFactory _factory, CounterDeltaEngine _engine
    - Constructor: new CounterDeltaEngine(_factory, NullLogger<CounterDeltaEngine>.Instance)
    - Constants for test data: DefaultOid, DefaultAgent, DefaultMetricName

    Each test method calls RecordDelta with synthetic values and asserts on:
    - Return value (true = delta emitted, false = baseline stored)
    - _factory.CounterRecords count
    - _factory.CounterRecords[N].Delta value
    - _factory.CounterRecords[N].MetricName, Oid, Agent, Source labels
  </implementation>
</feature>
