**ICounterDeltaEngine interface:**

```csharp
/// <summary>
/// Computes and records counter deltas. Maintains per-OID+agent state to track
/// previous cumulative values and per-device sysUpTime for reboot detection.
/// </summary>
public interface ICounterDeltaEngine
{
    /// <summary>
    /// Compute and record a counter delta. Returns true if a delta was emitted
    /// to the metric factory, false on first-poll baseline (stored but not emitted).
    /// </summary>
    /// <param name="oid">The SNMP OID string.</param>
    /// <param name="agent">Device name or IP (cache key component).</param>
    /// <param name="source">"poll" or "trap".</param>
    /// <param name="metricName">Resolved metric name from OID map.</param>
    /// <param name="typeCode">SnmpType.Counter32 or SnmpType.Counter64.</param>
    /// <param name="currentValue">Current cumulative counter value from SNMP GET.</param>
    /// <param name="sysUpTimeCentiseconds">
    /// Device sysUpTime in centiseconds, or null if unavailable.
    /// When null, current &lt; previous is conservatively treated as reboot.
    /// </param>
    bool RecordDelta(
        string oid,
        string agent,
        string source,
        string metricName,
        SnmpType typeCode,
        ulong currentValue,
        uint? sysUpTimeCentiseconds);
}
```

- `private readonly ConcurrentDictionary<string, ulong> _lastValues = new();` -- keyed by `"oid|agent"`

**RecordDelta method** -- implement all 5 paths:

```csharp
public bool RecordDelta(
    string oid,
    string agent,
    string source,
    string metricName,
    SnmpType typeCode,
    ulong currentValue,
    uint? sysUpTimeCentiseconds)
{
    var key = $"{oid}|{agent}";

    // Atomic first-poll detection using AddOrUpdate.
    // If key is new (first poll): store baseline, previousValue stays null.
    // If key exists (subsequent poll): capture previous, update to current.
    ulong? previousValue = null;
    _lastValues.AddOrUpdate(
        key,
        addValueFactory: _ => currentValue,
        updateValueFactory: (_, prev) =>
        {
            previousValue = prev;
            return currentValue;
        });

    // Check sysUpTime for reboot detection BEFORE computing delta.
    // sysUpTime is per-device (agent), not per-OID.
    bool sysUpTimeDecreased =
        sysUpTimeCentiseconds.HasValue &&
        _lastSysUpTimes.TryGetValue(agent, out var lastUpTime) &&
        sysUpTimeCentiseconds.Value < lastUpTime;

    // Always update sysUpTime cache (so next poll has fresh baseline).
    if (sysUpTimeCentiseconds.HasValue)
        _lastSysUpTimes[agent] = sysUpTimeCentiseconds.Value;

    // Path 5: First poll -- baseline stored, no emission.
    if (previousValue is null)
    {
        _logger.LogDebug(
            "First poll baseline stored: Oid={Oid} Agent={Agent} Value={Value}",
            oid, agent, currentValue);
        return false;
    }

    // Compute delta based on counter type and sysUpTime.
    double delta;

    if (currentValue >= previousValue.Value)
    {
        // Path 1: Normal increment.
        delta = currentValue - previousValue.Value;
    }
    else if (sysUpTimeDecreased)
    {
        // Path 3: Reboot detected (sysUpTime decreased). Use current as delta.
        delta = currentValue;
        _logger.LogInformation(
            "Reboot detected (sysUpTime decreased): Oid={Oid} Agent={Agent}",
            oid, agent);
    }
    else if (typeCode == SnmpType.Counter32)
    {
        // Path 2: Counter32 wrap-around at 2^32.
        // Cast previousValue to uint for correct 32-bit arithmetic domain.
        delta = (Counter32Max - (uint)previousValue.Value) + currentValue;
        _logger.LogDebug(
            "Counter32 wrap-around: Oid={Oid} Agent={Agent} Previous={Previous} Current={Current}",
            oid, agent, previousValue.Value, currentValue);
    }
    else
    {
        // Path 4: Counter64 current < previous, sysUpTime not decreased.
        // 2^64 wrap is unreachable in practice -- treat as reboot.
        // Also covers: sysUpTime unavailable (null) + current < previous = conservative reboot.
        delta = currentValue;
        _logger.LogInformation(
            "Reboot detected (counter decreased, no wrap): Oid={Oid} Agent={Agent} TypeCode={TypeCode}",
            oid, agent, typeCode);
    }

    // Clamp to non-negative (safety net -- counters should never produce negative deltas).
    delta = Math.Max(0.0, delta);

    // Record the computed delta to the snmp_counter instrument.
    _metricFactory.RecordCounter(metricName, oid, agent, source, delta);
    return true;
}
```

**CRITICAL implementation notes from CONTEXT.md and RESEARCH.md:**
- Cache key is `$"{oid}|{agent}"` -- pipe character does not appear in OID strings or device names (DELT-05).
- `ulong` for cache values, not `double` -- prevents precision loss for large Counter64 values (Pitfall 2).
- `ConcurrentDictionary.AddOrUpdate` for atomic first-poll detection (Pitfall 1) -- avoids race where two concurrent polls for the same OID+agent both try to set baseline.
- sysUpTime stored per-device (agent key), NOT per-OID -- one sysUpTime value shared across all OIDs for a device.
- Counter32 wrap: cast `previousValue` to `uint` before subtraction to stay in 32-bit domain (Open Question 2).
- sysUpTime `null` + current < previous = conservative reboot assumption (CONTEXT.md decision).
- First-poll logged at Debug (routine). Reboot logged at Information (notable operational event). Wrap logged at Debug (routine on high-traffic interfaces).
- `Math.Max(0.0, delta)` clamp at the end (safety net, not expected to trigger).

Required usings: `System.Collections.Concurrent`, `Lextm.SharpSnmpLib`, `Microsoft.Extensions.Logging`, `SnmpCollector.Telemetry`.
