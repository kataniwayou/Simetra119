**SnmpSource.cs** (in `src/SnmpCollector/Pipeline/`):
```csharp
namespace SnmpCollector.Pipeline;

/// <summary>
/// Identifies whether an SNMP OID event originated from a poll or a trap.
/// Used as the "source" label value on business metrics.
/// </summary>
public enum SnmpSource
{
    Poll,
    Trap
}
```

**SnmpOidReceived.cs** (in `src/SnmpCollector/Pipeline/`):
Sealed class (NOT record) implementing `MediatR.INotification`. Mutable so behaviors can enrich in-place.

Properties:
- `required string Oid { get; init; }` -- raw OID string from SNMP PDU
- `required IPAddress AgentIp { get; set; }` -- sender IP (`set` not `init` because trap path may update)
- `string? DeviceName { get; set; }` -- poll path sets at publish time; null for traps until resolved
- `required ISnmpData Value { get; init; }` -- SharpSnmpLib typed value wrapper
- `required SnmpSource Source { get; init; }` -- Poll or Trap
- `required SnmpType TypeCode { get; init; }` -- SharpSnmpLib type code for dispatch
- `string? MetricName { get; set; }` -- null until OidResolutionBehavior runs

Required usings: `Lextm.SharpSnmpLib`, `MediatR`, `System.Net`.

**PipelineMetricService.cs** (in `src/SnmpCollector/Telemetry/`):
Sealed class implementing `IDisposable`. Singleton service owning all 6 pipeline counter instruments.

Constructor takes `IMeterFactory meterFactory` and `IOptions<SiteOptions> siteOptions`:
- Create meter: `_meter = meterFactory.Create(TelemetryConstants.MeterName)`
- Store `_siteName = siteOptions.Value.Name`
- Create all 6 counters as `Counter<long>`:
  - `_published = _meter.CreateCounter<long>("snmp.event.published")`
  - `_handled = _meter.CreateCounter<long>("snmp.event.handled")`
  - `_errors = _meter.CreateCounter<long>("snmp.event.errors")`
  - `_rejected = _meter.CreateCounter<long>("snmp.event.rejected")`
  - `_pollExecuted = _meter.CreateCounter<long>("snmp.poll.executed")`
  - `_trapReceived = _meter.CreateCounter<long>("snmp.trap.received")`

Public methods (all add 1 with `site_name` tag):
- `IncrementPublished()` -- PMET-01
- `IncrementHandled()` -- PMET-02
- `IncrementErrors()` -- PMET-03
- `IncrementRejected()` -- PMET-04
- `IncrementPollExecuted()` -- PMET-05
- `IncrementTrapReceived()` -- PMET-06

Each method: `_counter.Add(1, new TagList { { "site_name", _siteName } })`

`Dispose()` calls `_meter.Dispose()`.

Required usings: `System.Diagnostics.Metrics`, `Microsoft.Extensions.Options`, `SnmpCollector.Configuration`.
