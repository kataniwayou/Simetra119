  **Negative assertion pattern for G2 (used in all 4 scripts):**
  After confirming G1 states, check that G2 has NO tier logs over a 15s window:
  ```bash
  # Wait 10s to accumulate several SnapshotJob cycles (1s interval = 10 cycles)
  sleep 10

  PODS=$(kubectl get pods -n simetra -l app=snmp-collector \
      -o jsonpath='{.items[*].metadata.name}' 2>/dev/null) || true
  G2_FOUND=0
  for POD in $PODS; do
      G2_LOGS=$(kubectl logs "$POD" -n simetra --since=15s 2>/dev/null \
          | grep "e2e-tenant-G2.*tier=" || echo "") || true
      if [ -n "$G2_LOGS" ]; then
          G2_FOUND=1
          log_info "SNS-BX: UNEXPECTED G2 tier log in pod ${POD}: ${G2_LOGS}"
          break
      fi
  done

  if [ "$G2_FOUND" -eq 0 ]; then
      record_pass "SNS-BX: G2 not evaluated (gate blocked)" "G2 tier logs absent in 15s window"
  else
      record_fail "SNS-BX: G2 not evaluated (gate blocked)" "G2 tier log found — gate did NOT block"
  fi
  ```
