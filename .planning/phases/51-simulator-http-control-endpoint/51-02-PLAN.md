**Dockerfile:** Add `EXPOSE 8080/tcp` line after the existing `EXPOSE 161/udp`:

```dockerfile

WORKDIR /app

COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

COPY e2e_simulator.py .

EXPOSE 161/udp
EXPOSE 8080/tcp

CMD ["python", "-u", "e2e_simulator.py"]
```

No other Dockerfile changes needed. The existing `pip install -r requirements.txt` will pick up aiohttp automatically.
