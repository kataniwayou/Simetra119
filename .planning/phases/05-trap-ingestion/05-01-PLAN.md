**DeviceChannelManager** (`src/SnmpCollector/Pipeline/DeviceChannelManager.cs`):
- `namespace SnmpCollector.Pipeline;`
- Sealed class implementing `IDeviceChannelManager`
- Constructor takes: `IDeviceRegistry deviceRegistry`, `IOptions<ChannelsOptions> channelsOptions`, `PipelineMetricService pipelineMetrics`, `ILogger<DeviceChannelManager> logger`
- At construction, iterate `deviceRegistry.AllDevices` and create one `Channel.CreateBounded<VarbindEnvelope>()` per device:
  - `BoundedChannelOptions` with capacity from `channelsOptions.Value.BoundedCapacity`
  - `FullMode = BoundedChannelFullMode.DropOldest`
  - `SingleWriter = false` (multiple UDP packets may arrive concurrently)
  - `SingleReader = true` (one consumer Task per device)
  - `AllowSynchronousContinuations = false` (listener must not block on consumer continuations)
- Use the `Channel.CreateBounded(options, itemDropped)` overload for drop tracking
- `itemDropped` callback: call `pipelineMetrics.IncrementTrapDropped(deviceName)`, maintain a per-device `int` drop log counter (Dictionary<string, int>, initialized at construction). Increment counter; if `count % 100 == 0`, log Warning: "Trap channel for {DeviceName} has dropped {Count} varbinds (capacity {Capacity})"
- Store channels in `Dictionary<string, Channel<VarbindEnvelope>>` keyed by device name (StringComparer.Ordinal)
- Store drop log counters in a separate `Dictionary<string, int>` (NOT ConcurrentDictionary — the itemDropped callback fires on the writer thread; with SingleWriter=false, multiple writers may call concurrently, so use `Interlocked.Increment` on a long array or accept approximate counts for the log threshold). Simplest correct approach: use `ConcurrentDictionary<string, long>` for drop counters, but since `Interlocked.Increment(ref dict[key])` won't compile, use a wrapper: `private sealed class DropCounter { public long Count; }` and do `Interlocked.Increment(ref _dropCounters[name].Count)`. This is lock-free and correct.
- `GetWriter(string deviceName)` returns `_channels[deviceName].Writer`
- `GetReader(string deviceName)` returns `_channels[deviceName].Reader`
- `DeviceNames` returns `_channels.Keys` as `IReadOnlyCollection<string>`
- `CompleteAll()` iterates all channels and calls `channel.Writer.TryComplete()`
- Startup log in constructor: `_logger.LogInformation("Device channels created for {Count} devices (capacity {Capacity})", _channels.Count, capacity)`
