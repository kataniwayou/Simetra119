**Class structure:**
- `namespace SnmpCollector.Services;`

- Constructor dependencies (inject via DI):

- Private fields:
  - `private readonly ConcurrentDictionary<string, byte> _seenDevices = new();` — first-contact tracking
  - `private readonly UserRegistry _userRegistry = new();` — SharpSnmpLib requires it even for v2c; create once, reuse across all datagrams

**CRITICAL ARCHITECTURAL CONSTRAINT: This class must NEVER reference ISender, IPublisher, IMediator, or MediatR. The listener only writes to channels via IDeviceChannelManager. Verify by ensuring no MediatR using directive exists in this file.**

**ExecuteAsync(CancellationToken stoppingToken):**
1. Create `IPEndPoint` from `_listenerOptions.BindAddress` (IPAddress.Parse) and `_listenerOptions.Port`
2. Create `using var udpClient = new UdpClient(endpoint);`
3. Log at Information: `"Trap listener bound to UDP {Port}, monitoring {DeviceCount} devices"` with port and `_deviceRegistry.AllDevices.Count`
4. Enter `while (!stoppingToken.IsCancellationRequested)` receive loop:
   - `var result = await udpClient.ReceiveAsync(stoppingToken);`
   - Call `ProcessDatagram(result);` (synchronous — TryWrite is sync)
5. Exception handling in the loop:
   - `catch (OperationCanceledException) when (stoppingToken.IsCancellationRequested)` → `break;`
   - `catch (SocketException ex)` → LogWarning `"Socket error receiving SNMP trap: {Message}"`
   - `catch (Exception ex)` → LogWarning `"Malformed SNMP packet from unknown source: {Message}"`

**StopAsync override:**
Override `StopAsync(CancellationToken cancellationToken)` to call `_channelManager.CompleteAll()` after `base.StopAsync(cancellationToken)`. This marks all channel writers as complete, allowing ChannelConsumerService to drain remaining items via ReadAllAsync and finish naturally. Order: base.StopAsync first (cancels ExecuteAsync receive loop), then CompleteAll (signals consumers to drain).

**ProcessDatagram(UdpReceiveResult result):**
1. Parse messages:
   ```csharp
   IList<ISnmpMessage> messages;
   try
   {
       messages = MessageFactory.ParseMessages(result.Buffer, 0, result.Buffer.Length, _userRegistry);
   }
   catch (Exception ex)
   {
       _logger.LogWarning("Malformed SNMP packet from {SourceIp}: {Error}", result.RemoteEndPoint.Address, ex.Message);
       return;
   }
   ```
2. For each message, check `if (message is not TrapV2Message trapV2) continue;`
3. Extract sender IP: `var senderIp = result.RemoteEndPoint.Address;` — call `.MapToIPv4()` if RemoteEndPoint may be IPv6-mapped
4. Extract community: `var receivedCommunity = trapV2.Community().ToString();`
5. Extract variables: `var variables = trapV2.Variables();`

**Step 1 — Device lookup (MUST come before auth because expected community string lives on DeviceInfo):**
```csharp
if (!_deviceRegistry.TryGetDevice(senderIp, out var device))
{
    var firstOid = variables.Count > 0 ? variables[0].Id.ToString() : "(no varbinds)";
    _logger.LogWarning("Trap from unknown device {SourceIp}, first OID: {Oid}", senderIp, firstOid);
    _pipelineMetrics.IncrementTrapUnknownDevice();
    continue;
}
```

**Step 2 — Community string auth (case-sensitive, RFC-compliant):**
```csharp
if (!string.Equals(receivedCommunity, device.CommunityString, StringComparison.Ordinal))
{
    _logger.LogWarning("Auth failure from {SourceIp} ({DeviceName}): received community '{ReceivedCommunity}'",
        senderIp, device.Name, receivedCommunity);
    _pipelineMetrics.IncrementTrapAuthFailed();
    continue;
}
```

**Step 3 — First-contact logging:**
```csharp
if (_seenDevices.TryAdd(device.Name, 0))
{
    _logger.LogInformation("First trap received from {DeviceName} ({Ip})", device.Name, senderIp);
}
```

**Step 4 — Route each varbind to device channel:**
```csharp
var writer = _channelManager.GetWriter(device.Name);
foreach (var variable in variables)
{
    var envelope = new VarbindEnvelope(
        Oid: variable.Id.ToString(),
        Value: variable.Data,
        TypeCode: variable.Data.TypeCode,
        AgentIp: senderIp,
        DeviceName: device.Name);
    writer.TryWrite(envelope);
    // itemDropped callback in DeviceChannelManager handles counter + periodic warning
}
```
