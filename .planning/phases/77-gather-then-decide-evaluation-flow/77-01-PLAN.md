**New flow structure (follow the skeleton from RESEARCH.md exactly):**

1. **Pre-tier (NotReady):** Unchanged ŌĆö `if (!AreAllReady(tenant.Holders))` returns early with `RecordAndReturn(tenant, TenantState.NotReady, sw)`. NO percentage gauge calls. This is the ONLY early return.

2. **GATHER phase:**

   - `var staleTotal = CountStalenessEligibleHolders(tenant.Holders);`
   - `var isStale = staleCount > 0;`
   - If stale: set `resolvedViolatedCount = 0; resolvedTotal = 1; evaluateViolatedCount = 0; evaluateTotal = 1;` (skip resolved/evaluate ŌĆö stale data unreliable; use 1 for total to avoid special-casing in percentage calc, result will be 0.0)
   - If not stale: `resolvedViolatedCount = CountResolvedViolated(tenant.Holders); resolvedTotal = CountResolvedParticipating(tenant.Holders); evaluateViolatedCount = CountEvaluateViolated(tenant.Holders); evaluateTotal = CountEvaluateParticipating(tenant.Holders);`

3. **Tier 4 (command dispatch):** Keep the existing command dispatch loop structure (foreach cmd in tenant.Commands, suppression check, TryWrite to channel, count dispatched/suppressed/failed). Keep the existing _pipelineMetrics.IncrementCommandXxx calls (those are pipeline counters, NOT tenant counters ŌĆö they stay). Run dispatch when `isStale || AreAllEvaluateViolated(tenant.Holders)`. If neither stale nor all-evaluate-violated, set dispatchedCount = suppressedCount = failedCount = 0 (no dispatch needed ŌĆö Resolved or Healthy path).

   CRITICAL: Command dispatch only runs for Unresolved state. The Resolved path (all resolved violated, not stale) and Healthy path do NOT dispatch commands. Guard the dispatch loop:
   ```csharp
   var dispatchedCount = 0;
   var suppressedCount = 0;
   var failedCount = 0;

   if (isStale || AreAllEvaluateViolated(tenant.Holders))
   {
       // existing dispatch loop here (unchanged)
   }
   ```

4. **DECIDE phase (same priority order as v2.4):**
   ```csharp

   if (isStale)
       state = TenantState.Unresolved;
   else if (AreAllResolvedViolated(tenant.Holders))
       state = TenantState.Resolved;
   else if (AreAllEvaluateViolated(tenant.Holders))
       state = TenantState.Unresolved;
   else
       state = TenantState.Healthy;
   ```
   Re-call the existing bool helpers ŌĆö do NOT duplicate their logic.

5. **COMPUTE PERCENTAGES:**
   ```csharp
   var stalePercent     = staleTotal == 0 ? 0.0 : staleCount * 100.0 / staleTotal;
   var resolvedPercent  = resolvedTotal == 0 ? 0.0 : resolvedViolatedCount * 100.0 / resolvedTotal;
   var evaluatePercent  = evaluateTotal == 0 ? 0.0 : evaluateViolatedCount * 100.0 / evaluateTotal;

   var dispatchedPct    = cmdTotal == 0 ? 0.0 : dispatchedCount * 100.0 / cmdTotal;
   var failedPct        = cmdTotal == 0 ? 0.0 : failedCount * 100.0 / cmdTotal;
   var suppressedPct    = cmdTotal == 0 ? 0.0 : suppressedCount * 100.0 / cmdTotal;
   ```

6. **SINGLE EXIT (record all 6 gauges + state + duration):**
   ```csharp

   _tenantMetrics.RecordMetricResolvedPercent(tenant.Id, tenant.Priority, resolvedPercent);

   _tenantMetrics.RecordCommandDispatchedPercent(tenant.Id, tenant.Priority, dispatchedPct);
   _tenantMetrics.RecordCommandFailedPercent(tenant.Id, tenant.Priority, failedPct);
   ```
