## Objective

Rename all references to `tenantvector.json` / `simetra-tenantvector` to `tenants.json` / `simetra-tenants` across the entire codebase. This includes the C# constants, the `TenantVectorOptions.SectionName` (from `"TenantVector"` to `"Tenants"`), the local dev config file, K8s manifests, deployment volume projections, and all E2E test scripts and fixtures.

Purpose: Align the tenant config file naming to the new consistent convention. This is a mechanical rename — no logic changes.
Output: All files referencing the old name are updated; old-named files are git-moved to new names.
