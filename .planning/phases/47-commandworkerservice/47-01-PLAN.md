    **CommandChannel.cs**: Sealed class implementing ICommandChannel. Constructor creates `Channel.CreateBounded<CommandRequest>` with:

    - SingleWriter: false (multiple SnapshotJob tenant evaluations may write concurrently)
    - SingleReader: true (only CommandWorkerService reads)
    - AllowSynchronousContinuations: false
    - NO itemDropped callback (DropWrite mode does not invoke callbacks)

    Do NOT use `BoundedChannelFullMode.Wait` or `DropOldest`. The anti-patterns section in RESEARCH.md explains why.
