**Scenario 1: Add new device**
1. Add TEST-DEV-01 with unreachable IP (proves job creation without needing a real device):
```bash
kubectl get cm simetra-devices -n simetra -o json | \
  jq '.data["devices.json"] |= (fromjson | . + [{"Name":"TEST-DEV-01","IpAddress":"10.99.99.99","Port":161,"CommunityString":"Simetra.TEST-DEV-01","MetricPolls":[{"IntervalSeconds":30,"Oids":["1.3.6.1.4.1.47477.10.99.1.0"]}]}] | tojson)' | \
  kubectl apply -f -
```
2. Wait 10 seconds for watch event + processing
3. Check pod logs for device reload:
```bash
kubectl logs -n simetra -l app=snmp-collector --since=1m | grep "Device reload complete"
```
Expected: Lines showing `Device reload complete: 3 devices` (was 2, now 3)
4. Check logs for scheduler reconciliation:
```bash
kubectl logs -n simetra -l app=snmp-collector --since=1m | grep "Poll scheduler reconciled"
```
Expected: Lines showing `+1 added` in the reconcile summary

**Scenario 2: Change poll interval**
1. Change OBP-01 poll interval from 10s to 30s:
```bash
kubectl get cm simetra-devices -n simetra -o json | \
  jq '.data["devices.json"] |= (fromjson | map(if .Name == "OBP-01" then .MetricPolls[0].IntervalSeconds = 30 else . end) | tojson)' | \
  kubectl apply -f -
```
2. Wait 10 seconds
3. Check logs for reschedule:
```bash
kubectl logs -n simetra -l app=snmp-collector --since=1m | grep "Poll scheduler reconciled"
```
Expected: Lines showing `~1 rescheduled` in the reconcile summary
4. Verify OBP-01 metrics still flowing in Prometheus (polling continues at new interval):
```bash
curl -s -G "http://localhost:9090/api/v1/query" \
  --data-urlencode 'query=snmp_gauge{device_name="OBP-01",source="poll"}' | \
  jq '.data.result | length'
```
Expected: > 0

**Scenario 3: Remove device**

```bash

  kubectl apply -f -
```
2. Wait 10 seconds
3. Check logs for reload and job removal:
```bash
kubectl logs -n simetra -l app=snmp-collector --since=1m | grep "Device reload complete"
```
Expected: Lines showing `Device reload complete: 2 devices` (back to original count)
4. Check reconcile log:
```bash
kubectl logs -n simetra -l app=snmp-collector --since=1m | grep "Poll scheduler reconciled"
```
Expected: Lines showing `-1 removed` in the reconcile summary
